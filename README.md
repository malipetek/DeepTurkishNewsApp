# My Favorite Starter Template for any front end project

## Get started
  ```bash
  npx degit malipetek/SvelteKit-Flowbite-starter
  ```

### Install dependencies
```bash
  npm install
  ```

Or:
```bash
yarn install
```
Or:
```bash
pnpm install
```
---

### Start dev server
```bash
npm run dev
```

Or:
```bash
yarn dev
```
Or: 
```bash
pnpm run dev
```

---
